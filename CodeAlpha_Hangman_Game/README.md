# CodeAlpha Python Internship Project

## Project Name

Hangman Game (Python)

## Description

This project is a simple Hangman Game built using Python.
The user has to guess the hidden word by entering letters.
The game shows correct letters and decreases chances for wrong guesses.

## Features

* Random word selection
* User input for guessing letters
* Displays progress of guessed word
* Limited attempts to guess

## Technologies Used

* Python

## How to Run

1. Download the file
2. Run in terminal:

```
python hangman.py
```
